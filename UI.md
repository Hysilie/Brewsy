# UI / THEME — Cozy Pastel Direction

## Objectif visuel
Le design actuel est trop sombre et trop “dashboard admin”.
Je veux un rendu :
- cozy
- pastel
- doux
- chaleureux
- inspiré papeterie / bougies / nuages
- très lisible malgré des inputs

Le fond doit être clair, les cards doivent “flotter”, et les couleurs vives doivent être utilisées avec parcimonie.

---

### Fond global
- Background principal : `#FBF7F2`
- Variante dégradé (très subtil) :
  - top : `#FBF7F2`
  - bottom : `#F6EFE6`

## Thème clair (principal)

### Fond global
- Background principal : `#FBF7F2`
- Variante dégradé (très subtil) :
  - top : `#FBF7F2`
  - bottom : `#F6EFE6`

---

### Cards
- Fond carte principal : `#FFFFFF`
- Fond carte secondaire / section : `#FFF6F1`
- Bordure : `#EADFD8`
- Ombre :0 8px 20px rgba(0, 0, 0, 0.04)

- Border radius : `20px` à `24px`

---

### Texte
- Texte principal : `#2F2B2A`
- Texte secondaire : `#7A6F6B`
- Texte discret / placeholder : `#9C8F8A`

---

## Accents pastel (à utiliser avec modération)

Ne jamais utiliser plus d’1 accent fort par carte.

- Rose pêche : `#F4B6A8`
- Beurre doux : `#F6E3A3`
- Lavande : `#C9C3E6`
- Sauge : `#BFD8C2`
- Bleu ciel : `#AFC8E8`

### Usage recommandé
- chiffres clés
- badges
- highlights légers
- focus input

---

## Sidebar (thème clair)

- Fond : `#F1E9E1`
- Item actif :
- fond : `#F4B6A8`
- texte : `#2F2B2A`
- Item inactif :
- texte : `#6F6460`
- Icônes : même couleur que le texte (pas multicolores)

La sidebar doit être calme et ne pas voler l’attention.

---

## Dashboard — couleurs recommandées

- Valeur totale des stocks → `#F4B6A8`
- Transformations actives → `#C9C3E6`
- Transformations prêtes → `#BFD8C2`

Le reste du texte reste neutre.

---

## Inputs (cozy, non agressifs)

- Fond : `#FFFFFF`
- Bordure : `#EADFD8`
- Texte : `#2F2B2A`
- Placeholder : `#9C8F8A`
- Focus :
- border : accent pastel (selon le contexte)
- shadow très légère (pas bleue)

---

## Thème sombre (cozy night, PAS noir)

### Fond
- Background : `#1C1A22`

### Cards
- Fond carte : `#262330`
- Bordure : `#332F3D`

### Texte
- Texte principal : `#F3EEF4`
- Texte secondaire : `#B7AEB9`

### Accents (désaturés)
- Rose nuit : `#D9A1A6`
- Lavande nuit : `#9E99C9`
- Sauge nuit : `#8FB7A0`

⚠️ Le dark mode doit rester doux, jamais contrasté brutalement.

---

## Règles UX importantes

- transitions douces (`200ms ease-out`)
- hover léger sur les cards (translateY(-2px))
- pas de couleurs saturées type “bleu primaire”
- contraste suffisant mais jamais agressif
- priorité à la sensation “calme” et “confort”

---

## Résumé pour implémentation
- Fond clair crème
- Cards blanches / ivoire
- Accents pastel doux
- Sidebar claire
- Dark mode aubergine doux
- Zéro couleur “dashboard pro”
